I am a first-year engineering student and I participated in Codejam 2020

This was my first codejam and my first coding competition too

my review

choice of programming language

I used Node.Js for the competition as I am pretty fluent in JavaScript

Though JavaScript is a very popular programming language, within a few problems I realised that it's not very perfect for competitive programming

quality of problems

the qualification round was extremely easy and anyone with decent programming knowledge would qualify easily

why did I quit

since, this was my first attempt in competitive programming, I was not very prepared for algorithm stuff

this, along with many other factors was the main reason of my failure

lessons learned

  • Use a better programming language like C++
  • Be more prepared with algorithms
  • Think efficiently
  • improve speed

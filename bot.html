TheAbbie: Bot

By Abhishek Chaudhary

Published 2020-06-26

TheAbbie Bot is TheAbbie's Data Extraction Bot, It extracts Data from web and keeps them for everyone to use, It is an unethical bot, It doesn't respect robots.txt nor any other conventions, If the data is public (sometimes private), It will be crawled, To prevent it just pray that it doesn't find you
